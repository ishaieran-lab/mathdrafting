# Changelog

## [v11] - 2026-04-10
### Initial project setup
- Organized project into professional folder structure
- Source file: `src/app/index.html`
- Archived original to `history/v11/worksheet-builder.html`
- Initialized git repository

### Features in this version
- Canvas-based worksheet builder
- Shape tools: rectangles, circles, triangles, polygons, trapezoids, lines, arrows
- Math tools: coordinate axes, function graphs, parabolas, angles, tangent lines
- Text & equations: inline editing, fractions, powers, MathJax rendering
- Export: PNG download, PDF/print
- Languages: Hebrew (RTL), English, German
- Themes: dark mode with color-coded accents
